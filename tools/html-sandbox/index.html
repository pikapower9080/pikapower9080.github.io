Embed An Interactive Code Snippet

  1. Head over to gist.github.com
  2. Sign up for a GitHub account if you don't have one already.
  3. Create a new gist called (whatever).html
  4. Paste your code into the gist.
  5. Important: make sure that you click on the dropdown and select create public gist. Otherwise it won't work.
  6. Make note of the url. It should look something like gist.github.com/username/id and the id should be some random letters and numbers
  7. Copy the id. To test, go to pikapower9080.github.io/tools/html-viewer/embed?code=(your id here) and see if it worked.
  8. If so, embed that same url into your site. Use the iframe element if you are using html.
