Learn HTML

HTML or Hyper Text Markup Language is the programming language of the web. It allows you to create basic web pages with headings, text, images, and more. You can use the HTML viewer button at the bottom right of this page to test out HTML code on the fly. You can also interact with any of the code examples here by changing the code and seeing what happens. Just keep in mind that if you want to do some actual work, the preview code won't save, but the button in the bottom right will. Every good HTML file starts out with a basic shell. Let's walk through the components. The first line <!DOCTYPE html> tells the computer that this is an HTML document. The second line <html> is a tag. Tags open with <tagname> and most of the time get closed by </tagname>. The html tag contains everything in the document. Next comes the <head> tag. This tag basically contains the settings for your site. You won't find any images or text in this part. The <title> tag can be found inside of the head tag. The title tag contains the page's title which shows up in the browser tab.

Last is the most important part. The <body> tag. This is the tag that contains all of your text, images, heading, etc. Without the body tag, your HTML document would be blank. Let's create a <h1> tag inside of the body tag. (You can do this in the HTML viewer or in the sample above) Type whatever text you want into the tag and then remember to close it with </h1>. You should now have a boring looking heading at the top left of your preview.
