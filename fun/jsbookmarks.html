Javascript Bookmarks

Bookmarks are useful for all kinds of things, but one thing they're useful for is javascript!
Javascript is the language of the web. It's what makes webpages interactive, and what makes the Google X Website so annoying!
But jokes aside, javascript bookmarks can be super useful.

So let's figure out how to make them!

Javascript bookmarks are made just like any other bookmarks. I'm only going to show you how to do it in the chrome browser because I mean who uses anything else and I'm too lazy anyway.

Add bookmarks in the chrome browser

To add a bookmark in the chrome browser, open a new tab and type chrome://bookmarks
This will take you to a page with a list of your bookmarks. Next, click on the three dots in the top right and select "Add new bookmark"

Now in the dialog that shows up, enter the name that you want for your bookmark. In the url field instead of entering a real url, any javascript bookmark urls will start with javascript:
So, as a demonstration, type in javascript:alert("Hello world!"). Press done and there you have it! A working chrome javascript bookmark!


Alright, so now you have your Hello world javascript bookmark, let's test it out! Go to any page that's not your new tab or something like that (like this one) and click on your bookmark in the bookmarks bar. If everything worked out, it should look something like this.

Edit page

You might have heard of your browser's "Inspect element" feature. This allows you to edit a webpage, but just until you reload the page. The catch is that you have to know a little bit of html.
This bookmark pretty much lets you do that, but without the html part. At a super basic level, it turns everything into a text box that you can edit. The downside is that it's way less powerful than the good old Inspect Element.

Code:

javascript:document.body.contentEditable = "true";document.body.designMode='on'; void 0

Move page to a new window

This bookmark will move the current page into a new popup window, which is pretty neat.

Code:

javascript:window.open("#", "Popup Window", "width=1000,height=800")
