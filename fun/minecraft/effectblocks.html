Effect blocks datapack

This minecraft datapack gives the player certain effects when they stand on certain blocks

How to use

It's pretty simple after you install it into your world, the player will gain effects when standing on certain blocks.

Here is a video showcasing that.


Missed the video? Here's a catchup
  1. When you stand on magenta glazed terricotta you get speed
  2. When you stand on emerald blocks you get jump boost
  3. When you stand on red glazed terricotta you can't jump
  4. When you stand on coal you slowly take wither damage
  5. When you stand on lodestones you get slowness
  6. When you stand on crying obsidian you get speed and jump boost for 5 seconds after you leave the block

How to install

There are multiple ways to install, if one way doesn't work for you, check the other.

Find your world folder in game

The option is usually the best, just launch minecraft, then create a new world or find an old one.

Then if you're in the world, close out of it and click the singleplayer tab again if you need to so you should now be on the world selection screen

Next click on the world you chose in step one. Don't play the world just select it

Now in the bottom bar click edit and in that menu click open world folder.

Now a folder for your world should be opened. Locate the datapacks folder inside there.

Download the datapack and drag it over into the datapacks folder

And you're done!

Find your world folder via your minecraft folder

This one is generally a bit harder because it varries between computers but here's the gist:

Windows:

Press windows key + r. A window should open called "run" with a prompt to enter a folder path.

Type in %appdata% and then press enter. A folder should open up with a bunch of folders. Each of these is for a different app

Find .minecraft (show hidden folders if you dont see it if that won't work type %appdata%\.minecraft instead)

Now you should be at a folder with various things like "saves" and "resource packs". This is your minecraft folder. It has all minecraft related files.

Now locate the saves folder and select a world inside of there. Open up that folder and find "datapacks".

Download the datapack and drag it over into the datapacks folder

Mac:

Make sure you're not focused on any application (click on your desktop and you should see finder next to the apple logo in the top)

Press Command + Shift + G and type in /Library/Application Support/minecraft (if you get an error, try /Users/YOURUSERNAMEHERE/Library/Application Support/minecraft)

Now you should be at a folder with various things like "saves" and "resource packs". This is your minecraft folder. It has all minecraft related files.

Now locate the saves folder and select a world inside of there. Open up that folder and find "datapacks".

Download the datapack and drag it over into the datapacks folder
