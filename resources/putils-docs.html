putils Documentation

putils (or Pika's Utils) is a basic library that makes your vanilla javascript life easier. It includes functions that do things like execute code from a string or pick a random element from an array.

Installing

putils is meant to be used in a browser enviorment and not something like node.js. This means that there is no package on npm. Instead, it is imported using the <script> html tag. To install, add the following to your <head>.

<script src="https://cdn.jsdelivr.net/gh/pikapower9080/pikapower9080.github.io/assets/modules/utils.js"></script>

or you can load it from github if you are so inclined.

<script src="https://raw.githubusercontent.com/pikapower9080/pikapower9080.github.io/main/assets/modules/utils.js"></script>

Functions

putils has many functions you can use to make your life easier once you've loaded the script.

Quality of Life

Nobody wants to write out document.getElementById() so instead with putils you can just use g(elementId) or getElement(elementId) as a shortcut.

// Dont do this:
document.getElementById("myElement")
// Do this!
g("myElement")

Dialogs

The normal alert() and confirm() functions are super boring. So why not use putils.alertModal() instead! This will create a fancier alert dialog box with whatever message you want. You can also use HTML formatting!

putils.alertModal(<message>, ["optionalArrayOfClassNames", "forTheDialogElement", ["soYouCanStyleIt"])

A confirm box is more complicated but I think you're awesome and you can do it!

putils.confirmModal((result) => { 
    // Callback function that runs after something is selected
    console.log(result)
}, "Your confirmation message that supports <strong>HTML</strong> formatting!", <Button 1 text (default: OK)>, <Button 2 text (Default: Cancel)>, <OK Button CSS EX:"color: red;">, <Cancel Button CSS (you get the idea)>, ["listOfClassNames"])

All of those many options are optional except for the callback. If you for some reason don't want callback, just set it to () => {}.

Lastly, If you've ever wanted to add a toast to your website, it's super simple!

putils.toast("Message", 5000 (lifetime in milliseconds), true, <classNames array>)

The true (third argument) tells if the toast should use default styling or not. If you're going to use just your own amazing styles you can do so with the .putils-toast class or whatever else you added in the array. By the way, here's the default styles:

background-color: rgb(62, 62, 62); color: white; font-family: sans-serif; font-family: 13px; position: absolute; bottom: 0; left: 50%; margin-bottom: 35px; padding: 5px; transform: translateX(-50%); border-radius: 3px; opacity: 90%;

Pro Tip: Multiply the amount of seconds you want it to stay on screen by 1000 to get the number of milliseconds. So if I wanted it to stay live for 6 seconds I would put 6000 in the lifetime argument.

By the way, you can style the modals with .putils-modal, .putils-modal-alert, and .putils-modal-confirm.

More stuff

To pick a random item in an array try the following:

var myArray = ["John", "Jack", "Billy", "Paul", "Carrie"]
var randomElement = putils.array.pickrandom(myArray)
console.log(randomElement)
// Result might look something like: Billy

To execute code from a string use:

var myCodeString = "console.log('Hello from a string!')"
putils.execute(myCodeString)

Or make a function out of it:

var myCodeString = "console.log('Hello from a string!')"
var myFunction = putils.makeFunctionFromString(myCodeString)
myFunction()

To make a prompt that the user has to respond to (it's a little mean), use:

putils.forcePrompt("MWAHAHAHAHAHAHA YOU MUST ENTER SOMETHING INTO THIS PROMPT", "ok dude (default response)", "This will show up when you enter something that isn't valid. (NICE TRY IDIOT)")

This will create a prompt with the prompt() function that will just keep bugging the user until they enter something that isn't the cancel button or an empty string or a bunch of spaces.

To create an element in the body with one line of code do:

var myElement = putils.createElementInBody("div")

To create an element in another element with one line of code do:

var myElement = putils.createElementInBody("div")
            var myElement2 = putils.createElementIn("p", myElement)

To pick a random number the easy way do:

putils.randomInt(1, 10)
// Instead of
Math.floor(Math.random() * 10) // which you can't even set a min value for
